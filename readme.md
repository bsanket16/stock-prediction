# Stock Prediction
Streamlit Web App for Stock Prediction

## Dependencies

Run and install all libraries

```bash
pip install streamlit yfinance fbprophet plotly
```

## Development & Production

Use the streamlit to run the web app

```bash
streamlit run main.py
```
